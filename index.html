Что такое Gaming Zone?

Все просто - это приложения с простыми играми

  • Это приложение с играми
  • Мы реализовали только 3 игры, в планах будет больше

Что есть в нашем проекте:

  1. Регистрация, вторизация...
  2. 3 реализованные игры
  3. Уровни доступа
  4. Свой собственный
    • домен,
    • хостинг
    • smtp-сервер (email)
  • Регистрация, авторизация, смена пароля
  • 3 игры, одна из них мультиплеер, 1 в планах
  • Администратор, модератор, пользователь - может менять админ
  • бан - запрещает пользователю играть - может банить модератор
  • Свое доменно имя, сбросс пароля по email

Регистрация

Для регистрации необходимо ввести
  • Имя пользователя
  • E-mail
  • Пароль
  • Авто-вход после регистации

Выход

  • Для выхода просто нужно нажать на кнопку  Logout

Авторизация

  • Авторизация происходит по логину и паролю
  • После авторизации в coocie сохраняется JWT токен

Сброс пароля

  • Нажать Forgot password
  • Ввести имя пользователя или email
  • На почту придет письмо со ссылкой
  • Перейти -> вести новый пароль
  • Письмо с новым, введенным паролем

Изменение пароля

  • также можно изменить пароль в кабинете пользователя
  • введя старый и новый пароли
